# Chợ phiên Vân Hồ - Website triển lãm số V2

## File chính
- `index.html`: Trang người dân xem khi quét QR.
- `data.js`: Nơi quản lý tiêu đề, khu vực, gian hàng, danh sách ảnh.
- `admin.html`: Trang hỗ trợ tạo/copy file `data.js`.
- `images/`: Thư mục ảnh.

## Cách thay ảnh thật
1. Chuẩn bị ảnh thật, đổi tên theo mẫu: `01.jpg`, `02.jpg`, `03.jpg`.
2. Đưa vào đúng thư mục: `images/cong-cho`, `images/nong-san`, `images/am-thuc`, `images/van-hoa`.
3. Upload lên GitHub và Commit.
4. Nếu thêm ảnh mới ngoài danh sách đang có, sửa `data.js` hoặc dùng `admin.html` để tạo nội dung mới.
