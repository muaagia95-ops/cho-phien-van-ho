Quản trị ảnh - Chợ phiên Vân Hồ

Trang này giúp tạo lại nội dung file data.js. Vì GitHub Pages là website tĩnh nên không thể tự lưu ảnh lên GitHub; sau khi tạo xong, Nhị ca copy nội dung data.js và upload ảnh theo đúng đường dẫn.

1. Cách thay ảnh nhanh nhất

Đặt ảnh thật theo đúng thư mục và tên file, ví dụ:

images/cong-cho/01.jpg
images/cong-cho/02.jpg
images/nong-san/01.jpg
images/am-thuc/01.jpg
images/van-hoa/01.jpg

Sau đó vào GitHub → Add file → Upload files → kéo thư mục images lên → Commit changes.

2. Danh sách ảnh hiện tại

3. Tạo file data.js mới

Có thể sửa trực tiếp nội dung bên dưới, rồi bấm sao chép. Sau đó vào GitHub mở file data.js → Edit → dán thay toàn bộ → Commit.


4. Xem trước ảnh
